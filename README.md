# Personal Portfolio Website

A modern and interactive **Portfolio Website** built using **HTML, CSS, and JavaScript**. This website showcases personal skills, projects, and includes dynamic effects such as typing animation, tab switching, mobile menu toggle, and a Google Sheets powered contact form.

## 🌟 Features

- **Intro Loading Animation** – Fades out the intro section after a short delay.
- **Tab Switching Functionality** – Switch between different sections like About, Skills, and Projects.
- **Mobile Menu Toggle** – Open and close side menu on mobile devices.
- **Typing Effect** – Dynamic typing animation showcasing roles like "UI/UX Designer", "Developer", and "Programmer".
- **Google Sheets Contact Form** – Submits form data directly to Google Sheets and shows success/error messages.
- **Responsive Design** – Works well on desktop and mobile devices.
- **Clean UI/UX** – Modern layout with smooth animations and interactive elements.

## 🛠 Technologies Used

- HTML5 for structure
- CSS3 for styling and animations
- JavaScript (ES6) for interactivity
- Google Sheets API for contact form backend
